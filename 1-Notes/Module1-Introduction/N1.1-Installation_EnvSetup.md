<a id="check-existing-python"></a>
## 1Ô∏è‚É£ Check Existing Python  
Open **Command Prompt (cmd)** or **PowerShell** and run:  

```bash
python --version
```

- If you see Python 3.12.x or higher ‚Üí Skip to Step 4.
- If you see Python is not recognized or an older version ‚Üí Continue to Step 2.

---
<a id="download-python-official-source"></a>
## 2Ô∏è‚É£ Download Python (Official Source)

- Visit: [Download Python for Windows](https://www.python.org/downloads/windows/)
- Download the latest Windows installer (64-bit), e.g., Python 3.12.x or newer.

- Run the installer:  
  ‚úÖ Check "Add Python to PATH" before installing.  
  ‚úÖ Keep all defaults, then click **Install Now**.

---
<a id="verify-installation"></a>
## 3Ô∏è‚É£ Verify Installation  
Open cmd/PowerShell again and run:

```bash
python --version
# Example: Python 3.13.7

pip --version
# Example: pip 25.2 from .../site-packages/pip (python 3.13)
```

> If both commands work ‚Üí installation successful üöÄ

---
<a id="install-visual-studio-code-vs-code"></a>
## 4Ô∏è‚É£ Install Visual Studio Code (VS Code)
- Download: [Visual Studio Code for Windows](https://code.visualstudio.com/)
- Run the `.exe` installer and keep defaults.
- Once installed, open VS Code.
- Press `Win + S` ‚Üí Type "Visual Studio Code" ‚Üí Hit Enter.
- Optional (add code command to PATH):  
  In VS Code, press `Ctrl + Shift + P` ‚Üí type:

```bash
Shell Command: Install 'code' command in PATH
```
- Now you can open folders directly from terminal:
```bash
code .
```

---
<a id="install-python-extension-in-vs-code"></a>
## 5Ô∏è‚É£ Install Python Extension in VS Code
- Open VS Code.
- Click **Extensions** (left sidebar).
- Search: **Python**.
- Install the extension from Microsoft.

---
<a id="create-a-python-project-folder"></a>
## 6Ô∏è‚É£ Create a Python Project Folder
Create a folder for your course files:

```bash
mkdir Python_Course_By_Wiculty
```
- Open VS Code ‚Üí File ‚Üí Open Folder ‚Üí Select `Python_Course_By_Wiculty`

---
<a id="setup-github-integration-optional"></a>
## 7Ô∏è‚É£ Setup GitHub Integration (Optional)
- In VS Code, click **Source Control** (sidebar).
- Click **Publish to GitHub**.
- Sign in / create a GitHub account.
- Now you can push your code directly from VS Code.

---

> üéâ You‚Äôre Ready!  
> You can now start coding Python projects on Windows with VS Code. üöÄ

<a id="Check Existing Python"></a>
## 1Ô∏è‚É£ Check Existing Python

Open **Terminal** and run:

```bash
python3 --version
```

- **If you see Python 3.12.x or higher:**  
  _Skip to Step 4_

- **If you see Python 3.9.x (system default):**  
  _Continue to Step 2_

---
<a id="download-python-official-source"></a>
## 2Ô∏è‚É£ Download Python (Official Source)

- Visit: [Download Python for macOS](https://www.python.org/downloads/macos/)
- Download the latest **macOS 64-bit universal2 installer** (Python 3.12.x or newer).
- Double-click the `.pkg` file and follow the installer (keep defaults).

---

<a id="verify-installation"></a>
## 3Ô∏è‚É£ Verify Installation

Restart your terminal and check:

```bash
python3 --version
# Example: Python 3.13.7
```

Check the installation path:

```bash
which python3
# Example: /Library/Frameworks/Python.framework/Versions/3.13/bin/python3
```

Check pip (Python package manager):

```bash
pip3 --version
# Example: pip 25.2 from .../site-packages/pip (python 3.13)
```

---
<a id="install-visual-studio-code-vs-code"></a>
## 4Ô∏è‚É£ Install Visual Studio Code (VS Code)

- Download: [Visual Studio Code](https://code.visualstudio.com/)
- If `.dmg` file: Open ‚Üí drag **Visual Studio Code.app** to Applications.
- If `.zip` file: Extract ‚Üí drag to Applications.

**Open VS Code:**  
Press `Command + Space`, type `Visual Studio Code`.

**Add VS Code to PATH:**  
Press `Command + Shift + P` ‚Üí type:

```bash
Shell Command: Install 'code' command in PATH
```
‚Üí Hit Enter

Now you can open VS Code from terminal:

```bash
code .
```

**Adjust VS Code size:**  
View ‚Üí Appearance ‚Üí Zoom In / Zoom Out

---
<a id="install-python-extension-in-vs-code"></a>
## 5Ô∏è‚É£ Install Python Extension in VS Code

The Python extension provides:

- Interpreter selection
- Code running & debugging
- Formatting
- Jupyter Notebook support

**Steps:**

1. Open VS Code
2. Go to Extensions (left sidebar)
3. Search for **Python**
4. Install the extension from Microsoft

_Optional:_ Install **Jupyter** extension for notebooks.

---
