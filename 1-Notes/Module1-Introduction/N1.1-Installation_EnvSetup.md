<a id="verify-installation"></a>
## 3Ô∏è‚É£ Verify Installation  
Open cmd/PowerShell again and run:

```bash
python --version
# Example: Python 3.13.7

pip --version
# Example: pip 25.2 from .../site-packages/pip (python 3.13)
```

> If both commands work ‚Üí installation successful üöÄ

> ### What is PIP?
**`pip`** is the **package installer for Python**. It allows you to **install, update, and manage third-party Python packages** from the [Python Package Index (PyPI)](https://pypi.org/).

#### ‚úÖ Example: Installing a Package

To install the popular `numpy` library:

```bash
pip install numpy

# NumPy (Numerical Python) is a powerful library for:
- Numerical and mathematical operations
- Handling large multi-dimensional arrays and matrices
- Performing vectorized computations (much faster than loops)
- It‚Äôs the base layer for almost every data-related library ‚Äî like pandas, matplotlib, 
  scikit-learn, TensorFlow, etc.
```
#### ‚úÖ Python Package Index (PyPI): 
- PyPI (https://pypi.org/) is the official online repository where Python developers upload and 
    share their packages (libraries, frameworks, or modules).
- You can think of it as - "Play Store for Python packages".
- Every time you install something like numpy, or pandas ‚Äî it‚Äôs downloaded from PyPI.
- Example:
- üëâ Visit https://pypi.org/project/numpy/
You‚Äôll find the version history, documentation link, and install command (pip install numpy).

> #### üîë Key Points

- **Name**: `pip` stands for **"Pip Installs Packages"** (a recursive acronym).
- **Purpose**: Manages external Python libraries.
- **Default**: Included with Python 3.4 and newer.
- **Works in**: Terminal (Linux/Mac) or Command Prompt/PowerShell (Windows).

#### üõ† Common `pip` Commands

| Command | Description |
|--------|-------------|
| `pip install package-name` | Install a package |
| `pip uninstall package-name` | Uninstall a package |
| `pip list` | List all installed packages |
| `pip show package-name` | Show details of a package |
| `pip install -r requirements.txt` | Install packages listed in a file |

#### üí° Tips
- If you have multiple Python versions, use `pip3` for Python 3:  
  ```bash
  pip3 install package-name
  ```

## 4Ô∏è‚É£ Install Visual Studio Code (VS Code)
- Download: [Visual Studio Code for Windows](https://code.visualstudio.com/)
- Run the `.exe` installer and keep defaults.
- Once installed, open VS Code.
- Press `Win + S` ‚Üí Type "Visual Studio Code" ‚Üí Hit Enter.
- Optional (add code command to PATH):  
  In VS Code, press `Ctrl + Shift + P` ‚Üí type:

```bash
Shell Command: Install 'code' command in PATH
```
- Now you can open folders directly from terminal:
```bash
code .
```
**Adjust VS Code size:**  
View ‚Üí Appearance ‚Üí Zoom In / Zoom Out

<a id="install-python-extension-in-vs-code"></a>
## 5Ô∏è‚É£ Install Python Extension in VS Code

The Python extension provides:

- Interpreter selection
- Code running & debugging
- Formatting
- Jupyter Notebook support

**Steps**
- Open VS Code.
- Click **Extensions** (left sidebar).
- Search: **Python**.
- Install the extension from Microsoft.

> **Note:** You need to install the Python extension in VS Code because it doesn't support Python by default ‚Äî the extension adds essential features like IntelliSense, debugging, linting, and environment management.
