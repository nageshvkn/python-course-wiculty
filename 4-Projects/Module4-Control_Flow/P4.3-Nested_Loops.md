## ğŸ§© Mini Project-3: 2D Matrix Input and Sum

### ğŸ“‹ Objective
Accept a 3x3 matrix input from the user and compute the sum.

---
### âœ… Solution:
```python
matrix = []
for i in range(3):
    row = []
    for j in range(3):
        val = int(input(f"Enter element ({i},{j}): "))
        row.append(val)
    matrix.append(row)

# Calculate sum
total = 0
for i in range(3):
    for j in range(3):
        total += matrix[i][j]

print("Matrix:", matrix)
print("Sum of all elements:", total)
```
